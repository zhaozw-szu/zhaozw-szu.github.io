ANOMALYCLIP: OBJECT-AGNOSTIC PROMPT LEARNING FOR ZERO-SHOT ANOMALY DETECTION

ТЉўУдЂ

РђІ жЏХТаиТюгт╝ѓтИИТБђТхІ№╝ѕZSAD, Zero-shot anomaly detection №╝ЅжюђУдЂСй┐ућеУЙЁтіЕТЋ░ТЇ«У┐ЏУАїУ«Гу╗ЃуџёТБђТхІТеАтъІ№╝їС╗ЦСЙ┐тюеуЏ«ТаЄТЋ░ТЇ«жЏєСИГТ▓АТюЅС╗╗СйЋУ«Гу╗ЃТаиТюгуџёТЃЁтєхСИІТБђТхІт╝ѓтИИсђѓУ┐ЎТў»СИђСИфУЄ│тЁ│жЄЇУдЂуџёС╗╗тіАТЌХ№╝їтйЊУ«Гу╗ЃТЋ░ТЇ«ТЌаТ│ЋУ«┐жЌ«ућ▒С║јтљёуДЇжЌ«жбў№╝їСЙІтдѓ№╝їТЋ░ТЇ«жџљуДЂ№╝їСйєт«ЃТў»тЁиТюЅТїЉТѕўТђДуџё№╝їтЏаСИ║ТеАтъІжюђУдЂТјет╣┐т╝ѓтИИтюеСИЇтљїжбєтЪЪтЅЇТЎ»т»╣У▒АуџётцќУДѓ№╝їт╝ѓтИИтї║тЪЪтњїУЃїТЎ»уЅ╣ТђД№╝їтдѓу╝║жЎи/Уѓ┐уўцтюеСИЇтљїуџёС║ДтЊЂ/тЎет«ў№╝їтЈ»С╗ЦТўЙУЉЌСИЇтљїсђѓТюђУ┐Љ№╝їтцДтъІуџёжбётЁѕУ«Гу╗ЃуџёУДєУДЅУ»ГУеђТеАтъІ№╝ѕVLMs№╝Ѕ№╝їтдѓCLIP№╝їтюетїЁТІгт╝ѓтИИТБђТхІтюетєЁуџётљёуДЇУДєУДЅС╗╗тіАСИГТўЙуц║тЄ║С║єтЙѕт╝║уџёжЏХТаиТюгУ»єтѕФУЃйтіЏсђѓуёХУђї№╝їт«ЃС╗гуџёZSADТђДУЃйУЙЃт╝▒№╝їтЏаСИ║vlmТЏ┤тЁ│Т│еС║јтЅЇТЎ»т»╣У▒Ауџёу▒╗У»ГС╣Ѕт╗║ТеА№╝їУђїСИЇТў»тЏЙтЃЈСИГуџёт╝ѓтИИ/ТГБтИИТђДсђѓ

РђІ тюеТюгТќЄСИГ№╝їТѕЉС╗гС╗Іу╗ЇС║єСИђуДЇТќ░уџёТќ╣Т│Ћ№╝їтЇ│AnomalyCLIP№╝їСй┐CLIPтюеСИЇтљїжбєтЪЪуџёу▓ЙуА«ZSADсђѓAnomalyCLIPуџётЁ│жћ«УДЂУДБТў»тГдС╣аСИјт»╣У▒АТЌатЁ│уџёТќЄТюгТЈљуц║№╝їУ┐ЎС║ЏТЈљуц║ТЇЋУјитЏЙтЃЈСИГуџёСИђУѕгТГБтИИТђДтњїт╝ѓтИИТђД№╝їУђїСИЇу«АтЁХтЅЇТЎ»т»╣У▒АтдѓСйЋсђѓУ┐ЎСй┐тЙЌТѕЉС╗гуџёТеАтъІУЃйтцЪтЁ│Т│ет╝ѓтИИуџётЏЙтЃЈтї║тЪЪ№╝їУђїСИЇТў»т»╣У▒АуџёУ»ГС╣Ѕ№╝їС╗јУђїУЃйтцЪт»╣СИЇтљїу▒╗тъІуџёт»╣У▒АТдѓТІгтйњу║│ТГБтИИтњїт╝ѓтИИУ»єтѕФсђѓтюе17СИфуюЪт«ът╝ѓтИИТБђТхІТЋ░ТЇ«жЏєСИіУ┐ЏУАїуџётцДУДёТеАт«ъжфїУАеТўј№╝їAnomalyCLIPтюеСИЇтљїу╝║жЎиТБђТхІтњїтї╗тГдТѕљтЃЈжбєтЪЪуџёжФўт║дтцџТаиТђДу▒╗У»ГС╣ЅТЋ░ТЇ«жЏєуџёт╝ѓтИИТБђТхІтњїтѕєтЅ▓Тќ╣жЮбтЁиТюЅС╝ўУХіуџёжЏХжЋютц┤ТђДУЃйсђѓ

ТюгТќЄуџёСИ╗УдЂУ┤Ауї«тдѓСИІсђѓ

РђІ РђІ1. ТѕЉС╗гждќТгАТЈГуц║С║єтГдС╣ат»╣У▒АСИЇтЈ»уЪЦуџёТќЄТюгТЈљуц║уџёТГБтИИтњїт╝ѓтИИТў»СИђуДЇу«ђтЇЋУђїТюЅТЋѕуџётЄєуА«уџёZSADТќ╣Т│ЋсђѓСИјуЏ«тЅЇСИ╗УдЂСИ║т»╣У▒АУ»ГС╣Ѕт»╣жйљУђїУ«ЙУ«АуџёТќЄТюгТЈљуц║Тќ╣Т│Ћ (Jeong et al., 2023; Zhou et al., 2022b)уЏИТ»ћ№╝їТѕЉС╗гуџёТќЄТюгТЈљуц║тхїтЁЦТеАтъІУ»ГС╣ЅуџёСИђУѕгт╝ѓтИИтњїТГБтИИ№╝їтЁЂУ«Ит»╣У▒АТЌатЁ│№╝їт╣┐С╣ЅZSADТђДУЃйсђѓ
РђІ 2. уёХтљј№╝їТѕЉС╗гт╝ЋтЁЦС║єСИђуДЇТќ░уџёZSADТќ╣Т│Ћ№╝їуД░СИ║AnomalyCLIP№╝їтЁХСИГТѕЉС╗гтѕЕућеСИђСИфт»╣У▒АСИЇтЈ»уЪЦуџёТЈљуц║ТеАТЮ┐тњїСИђСИфgт▒ђжЃет╝ѓтИИТЇЪтц▒тЄйТЋ░№╝ѕтЇ│тЁет▒ђтњїт▒ђжЃеТЇЪтц▒тЄйТЋ░уџёу╗ётљѕ№╝ЅТЮЦтГдС╣ажђџућет╝ѓтИИтњїТГБтИИТЈљуц║сђѓтюеТГцУ┐ЄуеІСИГ№╝їAnomalyCLIPтюетЙѕтцДуеІт║дСИіу«ђтїќС║єТЈљуц║У«ЙУ«А№╝їт╣ХтЈ»С╗ЦТюЅТЋѕтю░т║ћућеС║јСИЇтљїуџёжбєтЪЪ№╝їУђїСИЇжюђУдЂТЏ┤Тћ╣тЁХтГдС╣атѕ░уџёСИцСИфТЈљуц║№╝їУ┐ЎСИјWinCLIPуГЅуј░ТюЅуџёТќ╣Т│ЋСИЇтљї№╝їтљјУђЁуџёТюЅТЋѕТђДтЙѕтцДуеІт║дСИіСЙЮУхќС║јт»╣ТЋ░уЎЙСИфТЅІтіет«џС╣ЅТЈљуц║уџёт╣┐Т│ЏтиЦуеІсђѓ
РђІ 3. т»╣ТЮЦУЄфСИЇтљїтиЦСИџтњїтї╗тГджбєтЪЪуџё17СИфТЋ░ТЇ«жЏєУ┐ЏУАїуџёу╗╝тљѕт«ъжфїУАеТўј№╝їAnomalyCLIPтюеТБђТхІтњїтѕєтЅ▓ТЮЦУЄфу╝║жЎиТБђТЪЦтњїтї╗тГдТѕљтЃЈжбєтЪЪуџёжФўт║дтцџТаиТђДу▒╗У»ГС╣ЅТЋ░ТЇ«жЏєуџёт╝ѓтИИТќ╣жЮбтЁиТюЅС╝ўУХіуџёZSADТђДУЃйсђѓ

тЏЙ2№╝џAnomalyCLIPуџёТдѓУ┐░сђѓСИ║С║єСй┐CLIPжђѓт║ћС║јZSAD№╝їAnomalyCLIPт╝ЋтЁЦС║єСИјт»╣У▒АТЌатЁ│уџёТќЄТюгТЈљуц║ТеАТЮ┐ТЮЦТЇЋУјиСИђУѕгуџёТГБтИИТђДтњїт╝ѓтИИТђД№╝їУђїСИЇу«Ат»╣У▒АуџёУ»ГС╣ЅтдѓСйЋсђѓуёХтљј№╝їТѕЉС╗гт╝ЋтЁЦС║єтЁет▒ђСИіСИІТќЄС╝ўтїќ№╝їт░єтЁет▒ђтњїу╗єу▓њт║дуџёт╝ѓтИИУ»ГС╣Ѕу║│тЁЦтѕ░т»╣У▒АСИЇтЈ»уЪЦуџёТќЄТюгТЈљуц║тГдС╣аСИГсђѓТюђтљј№╝їСй┐ућеТќЄТюгТЈљуц║У░ЃС╝ўтњїDPAM№╝їтюеCLIPуџёТќЄТюгтњїт▒ђжЃеУДєУДЅуЕ║жЌ┤СИГт«ъуј░ТЈљуц║тГдС╣асђѓ
